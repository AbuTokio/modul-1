Position

Was ist Position?

Mit der CSS-Eigenschaft position wird gesteuert, wo die Elemente auf einer Seite dargestellt werden.

Wie funktioniert das?

position legt fest, wie die Elemente in unserem Dokument positioniert werden. Die Eigenschaften sind: top, bottom, left und right.

position: static

Standardwert für jedes HTML-Element. Das Element befindet sich an seiner Standardposition.

static

position: relative

Das Element bleibt an seiner ursprünglichen Position, kann aber von dort aus mit top, left, usw. verschoben werden.

relative

position: absolute

Das Element orientiert sich am nächstgelegenen nicht-statischen Eltern-Element. Wenn es ein solches Element nicht gibt, dann orientiert es sich am HTML-Element body.

Best Practice: Das Eltern-Element sollte die Eigenschaft position: relative erhalten.

absolute

position: fixed

Das Element orientiert sich am Browserfenster. Es wird aus dem Dokumentfluss entfernt und bleibt beim Scrollen immer an seiner Position.

fixed

position: sticky

Das Element scrollt mit, bis eine seiner Kanten, die durch top, bottom, left und/oder right angegebenen Korordinaten erreicht hat.

sticky
